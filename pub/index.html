Popup Quiz Library API

options

All the global options available.

size

Default: /

Type: String

Description: A CSS selecor for the modal size. you can choose between:

  • 'small'
  • 'medium'
  • 'large'

color

Default: 'white'

Type: String

Description: A CSS selecor for the modal background color. The color type accepted are: hexadecimal, rgb, rgba.

question (required)

Default: /

Type: Object

Description: Let you customize the quiz with the following format. Hint is optional.

  • multiple choice
    •                             {
                                type: 'multiple-choice',
                                question: 'what is 1 + 1?',
                                answer: {
                                  options: ['1', '2', '3', '4'],
                                  answer: '2'
                                }
                              }
  • true or false
    •                             {
                                type: 'true-or-false',
                                question: 'I like CSC309',
                                answer: {
                                answer: 'true'
                                }
                            }
  • drag and drop
    •                             {
                                type: 'drag-drop',
                                question: 'Choose words that starts with A.',
                                answer: {
                                    options: ['apple', 'orange', 'animal', 'cow'],
                                    answer: ['apple', 'animal']
                                },
                                hint: 'One of the answers is apple'
                            }
  • flash card
    •                             {
                                type: 'flash-card',
                                question: 'The color of the sky?',
                                answer: 'blue',
                                hint: 'Look at the sky'
                            }
  • short answer
    •                             {
                                type: 'short-answer',
                                question: 'what is the color of orange?',
                                answer: {
                                  answer: 'orange'
                                }
                            }

rating

Default: true

Type: Boolean

Description: It decides the rating section at the end of the quiz.

Methods

methods to control the modal 
                modalInstance.makeModal()
                // create the basic modal structure

                modalInstance.closeModal()
                // close the modal

Getting Started

                const mg = new ModalGenerator({
                    size: size,
                    color: color,
                    questions: [
                      {
                        type: 'drag-drop',
                        question: 'Choose words that starts with A.',
                        answer: {
                          options: ['apple', 'orange', 'animal', 'cow'],
                          answer: ['apple', 'animal']
                        },
                        hint: 'One of the answers is apple'
                      },
                      {
                        type: 'flash-card',
                        question: 'The color of the sky?',
                        answer: 'blue',
                        hint: 'Look at the sky'
                      },
                      {
                        type: 'multiple-choice',
                        question: 'what is 1 + 1?',
                        answer: {
                          options: ['1', '2', '3', '4'],
                          answer: '2'
                        }
                      },
                      {
                        type: 'short-answer',
                        question: 'what is the color of orange?',
                        answer: {
                          answer: 'orange'
                        }
                      },
                      {
                        type: 'true-or-false',
                        question: 'I like CSC309',
                        answer: {
                          answer: 'true'
                        }
                      }
                    ],
                    rating: rating
                  });
                mg.makeModal();
